# netflix_ui


```dart
lib
├── main.dart
├── models // 화면에 필요한 필요한 sample data와 data model class 관리 folder
└── screens // 6개의 screen이 모여있는 folder
    ├── coming // 공개 예정 screen에 사용될 widget
    ├── contents_list // 저장된 contents list screen에 사용될 widget
    ├── detail // 영상 회차 및 상세정보 screen에 사용될 widget
    ├── home // home screen에 사용될 widget
    ├── profile // profile 선택 screen에 사용될 widget
    └── splash // splash screen에 사용될 widget
```
